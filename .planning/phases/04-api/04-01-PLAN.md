**Create internal/api/upscale_handler.go** with handlers:

1. **handleListUpscaleJobs** (GET /api/upscale/jobs)
   - Call database.ListJobs()
   - Return jsonOK with jobs array (empty array if none)

2. **handleCreateUpscaleJob** (POST /api/upscale/jobs)
   - Decode JSON body: `{"inputPath": "...", "preset": "fast|balanced|quality"}`
   - Validate inputPath exists on filesystem (os.Stat)
   - Validate preset is valid (fast, balanced, quality)
   - Generate outputPath: replace extension with `_4k.mkv` (use strings.TrimSuffix + filepath.Ext)
   - Call database.CreateJob()
   - Return jsonOK with created job (201 status via w.WriteHeader before jsonOK)

3. **handleGetUpscaleJob** (GET /api/upscale/jobs/{id})
   - Get id from chi.URLParam(r, "id"), parse with strconv.ParseInt
   - Call database.GetJob()
   - Return 404 if nil, jsonOK if found

4. **handleDeleteUpscaleJob** (DELETE /api/upscale/jobs/{id})
   - Get id from URL param
   - Get job first to check status
   - Return 404 if not found
   - Return 400 "cannot delete running job" if status == "running"
   - Call database.DeleteJob()
   - Return jsonOK with {"deleted": true}
