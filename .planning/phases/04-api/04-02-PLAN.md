Add a CancelJob method to the Worker struct that allows external cancellation of a running job:

```go
// CancelJob cancels a job by ID if it's currently running.
// Returns true if the job was cancelled, false if not running or different job.
func (w *Worker) CancelJob(id int64) bool {
    w.mu.Lock()
    defer w.mu.Unlock()
    
    if w.currentJob != id || w.cancelFunc == nil {
        return false
    }
    
    w.cancelFunc()
    return true
}
```

This method:
- Is goroutine-safe (uses existing mutex)
- Only cancels if the requested job is actually running
- Returns bool so caller knows if cancel succeeded
- Uses existing cancelFunc mechanism from handleShutdown pattern
