     -vf "libplacebo=w=iw*2:h=ih*2:custom_shader_path={shader}"

   - Starts process with `cmd.Start()`
   - For now, just drain stderr in a goroutine (actual parsing in Plan 02)
   - Waits with `cmd.Wait()`
   - On error or context cancellation: `os.Remove(job.OutputPath)` to clean up partial file
   - Returns `ctx.Err()` if context was cancelled, otherwise wraps ffmpeg error
