<task type="auto">
  <name>Task 1: Create progress parser</name>
  <files>internal/upscale/progress.go</files>
  <action>
Create `internal/upscale/progress.go` with:

1. Package-level regex: 
   ```go
   var progressRegex = regexp.MustCompile(`frame=\s*(\d+)\s+fps=\s*([\d.]+).*time=(\d+:\d+:\d+\.\d+)`)
   ```

2. `scanFFmpegLines(data []byte, atEOF bool) (advance int, token []byte, err error)` function:
   - Custom bufio.SplitFunc that handles both `\r` and `\n` line endings
   - FFmpeg uses `\r` for in-place progress updates
   - Look for `\r` first (via `bytes.IndexByte`), return token if found
   - Fall back to `\n` if no `\r`
   - Return remaining data at EOF

3. `parseTimeToSeconds(timeStr string) float64` function:
   - Parses "HH:MM:SS.ms" format to seconds
   - Example: "00:05:23.45" -> 323.45
   - Use `strings.Split` on ":" and "."
   - Handle errors gracefully (return 0 on parse failure)

4. `parseProgress(r io.Reader, totalDuration float64, jobID int64, cb ProgressCallback)` function:
   - Creates `bufio.Scanner` with `scanFFmpegLines` split function
   - For each line, applies `progressRegex.FindStringSubmatch`
   - Throttles updates: skip if `time.Since(lastUpdate) < time.Second`
   - Extracts frame (int), fps (float64), time (string)
   - Calculates percent: `(parseTimeToSeconds(time) / totalDuration) * 100`
   - Calls `cb(models.UpscaleProgress{...})` with extracted values
   - Handles nil callback gracefully (no-op)

Import: `bufio`, `bytes`, `io`, `regexp`, `strconv`, `strings`, `time`, `link-anime/internal/models`
